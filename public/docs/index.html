Introduction

Base URL: http://krisah.local

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Register User

This service used to register user. If everything is okay, you'll get a 200 OK response.

Otherwise, the request will fail with a 400 error, and a response listing the failed services.

Example request:
curl --request POST \
    "http://krisah.local/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"Jon Doe\",
    \"email\": \"jondoe@example.com\",
    \"password\": \"AlphaNumericWithSpecialChars\"
}"

const url = new URL(
    "http://krisah.local/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "name": "Jon Doe",
    "email": "jondoe@example.com",
    "password": "AlphaNumericWithSpecialChars"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());
$client = new \GuzzleHttp\Client();
$url = 'http://krisah.local/register';
$response = $client->post(
    $url,
    [
        'headers' => [
            'Content-Type' => 'application/json',
            'Accept' => 'application/json',
        ],
        'json' => [
            'name' => 'Jon Doe',
            'email' => 'jondoe@example.com',
            'password' => 'AlphaNumericWithSpecialChars',
        ],
    ]
);
$body = $response->getBody();
print_r(json_decode((string) $body));
